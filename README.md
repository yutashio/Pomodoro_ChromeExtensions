# Pomodoro Timer_Chrome拡張機能
Chrome拡張機能を作成しました。  
ポモドーロテクニックを使用して作業を効率的に進めましょう！

- 作業時間の初期時間は、「25分」です。(お好きなお時間に変更可能です。)
- 休憩時間は、「5分」です。(固定時間になります。)
- 各時間終了時には、音声が流れます(音声は「歓声」が流れます。)  
  ※スピーカボタンを押下することで音声を消音モードにすることも可能です。
